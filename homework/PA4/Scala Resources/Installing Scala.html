Although you can install a Scala IDE (Intellij), I recommend just installing the same command-line binaries that I'll be using in class (i.e. running "scalac" and "scala" in a shell). To install the command line binaries:

1. Go to https://scala-lang.org/download/ and scroll to the bottom of the page where it says "Other Resources".

2. Click on the file scala-2.13.1.tgz (which is listed for "Mac OS X, Unix, Cygwin") to download it.

3. Unzip the file and move the resulting folder (named scala-2.13.1) to a directory where you keep other executables, e.g. /usr/local/ .  That is, you should end up with a directory /usr/local/scala-2.13.1

4. Add "/usr/local/scala-2.13.1/bin" to your PATH.  You can do this, for example, by putting the following line in your .bash_profile file:

export PATH="/usr/local/scala-2.13.1/bin":$PATH

and then creating a new shell.
